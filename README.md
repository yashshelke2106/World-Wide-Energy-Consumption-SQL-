# 🌍 World Wide Energy – SQL Project

## 📌 Overview

This project analyzes global energy data using SQL, focusing on GDP, energy production, consumption, emissions, and population to generate meaningful insights.

---

## 🎯 Objectives

* Analyze energy production vs consumption
* Identify top countries by GDP
* Measure emissions by energy type
* Evaluate energy efficiency (per capita & per GDP)
* Study global trends over time

---

## 🗄️ Database

```sql
USE energydb2;
```

---

## 🧱 Tables Used

* `gdp_3`
* `production`
* `consumption`
* `emission_3`
* `population`
* `country`

---

## 📊 Key Analysis

* Top 5 countries by GDP (latest year)
* Production vs consumption balance
* Emissions by energy type
* Energy production per capita
* Consumption relative to GDP
* Global emissions trend
* Emission share by country
* Global averages (GDP, emission, population)
* Latest emissions by country
* Highest consumption per year (window function)
* Highest production per capita per year

---

## 🚀 How to Run

1. Create database `energydb2`
2. Import datasets into tables
3. Execute SQL queries
